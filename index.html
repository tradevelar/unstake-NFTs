UNSTAKE NFTs

Admin tool. Force-unstake every NFT in a VeChain staking pool. Direct exit(tokenId), multi-clause batches.

Configuration

Wallet

Not connected. VeWorld extension required.

Connect with the wallet that holds DEFAULT_ADMIN on the staking pool. Each exit(tokenId) sends the NFT back to its original ticketOwner automatically.

Step 1 · Test 1 NFT first

Pulls the first staked tokenId and signs a single exit(tokenId). If this lands, the loop in Step 2 will too. Costs one transaction's worth of VTHO.

Step 2 · Unstake loop

Enumerates the next batch of tokenIds via tokenOfOwnerByIndex, packs N exit(tokenId) clauses into one transaction. Repeats until the pool's NFT balance hits zero.
